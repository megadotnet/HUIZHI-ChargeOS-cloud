# 业务逻辑说明文档目录 (Documentation Directory)

本目录包含“慧知开源充电桩平台”的深度业务逻辑剖析文档，旨在帮助产品经理（PM）和非技术人员理解系统运作机制。

## 文档列表

### 1. [用户角色与用例 (Roles & Use Cases)](./01_roles_and_usecases.md)
   - **内容**：定义系统中的核心角色（平台运营、租户、车主、监管方等），并通过用例图展示各角色的核心诉求与功能权限。
   - **目标**：明确“谁在使用系统”以及“他们要做什么”。

### 2. [核心业务流 (Core Business Flow)](./02_core_business_flow.md)
   - **内容**：详细描述业务运转的关键路径，特别是“扫码充电”的全生命周期。包含正向流程（启动、充电、结算）和逆向流程（启动失败、退款）。
   - **核心图表**：时序图 (Sequence Diagram)、流程图 (Flowchart)。
   - **目标**：阐述“业务是如何流转的”以及“资金是如何结算的”。

### 3. [状态机设计 (State Machine Design)](./03_state_machines.md)
   - **内容**：针对系统中最核心的实体（订单、设备）进行状态流转分析。
   - **核心图表**：状态图 (State Diagram)。
   - **目标**：通过状态变化透视业务规则，明确“什么状态下能做什么操作”。

### 4. [系统部署与外部集成 (System Deployment & Integration)](./04_system_integration.md)
   - **内容**：展示系统的物理部署架构，以及与外部系统（硬件网关、支付渠道、监管平台）的集成边界。
   - **核心图表**：部署图 (Deployment Diagram)、架构图。
   - **目标**：界定“系统的边界在哪里”以及“如何与外部世界交互”。
